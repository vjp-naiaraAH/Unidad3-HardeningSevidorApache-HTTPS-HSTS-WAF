img24.png-> CSP (Content Security Policy) es un mecanismo de seguridad que limita los orígenes de scripts, estilos e imágenes en una aplicación web para evitar ataques como XSS. Para habilitarla añadimos en el fichero de configuración del sitio y lo qu ehago es añadir al fichero "./config/vhosts/default.conf" la siguiente configuración "<IfModule mod_headers.c>
    Header always set Content-Security-Policy "default-src 'self'; script-src 'self'  object-src 'none'; base-uri 'self'; frame-ancestors 'none'"
</IfModule>"
